# template-surat

Generator Surat & Dokumen Profesional Indonesia 🇮🇩

Pilih template, isi form, langsung print atau download PDF.

## Tech Stack
- TypeScript + Vite
- Vanilla CSS (Premium Light Theme)
- localStorage for data persistence

## Features
- 📝 5+ Template surat Indonesia
- 🔍 Search & filter by category
- 📄 Live preview
- 🖨️ Print / PDF export
- 💾 Save & edit draft
- 🔒 Freemium model (3 surat/bulan gratis)
